# Fixtures

Fixtures in this repository must be synthetic or sanitized only.

Allowed fixture content includes:

- invented job postings and company names;
- synthetic ATS/API responses;
- reduced schema examples that contain no personal data;
- sanitized adapter payloads with all identifying details removed;
- deterministic examples for source policy, extraction, scoring, and audit tests.

Do not commit real candidate data, resumes, cover letters, screenshots, browser traces, raw scraped pages, secrets, cookies, tokens, private job-board responses, or application form contents.

When a fixture is derived from a real source, remove names, emails, phone numbers, addresses, account identifiers, URLs that reveal private context, and any employer or candidate-specific facts before committing it.
